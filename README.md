# Lodestar Strategy Group Website

A professional campaign strategy and consulting website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Product showcase and pricing
- Contact form
- Secure checkout modal
- Smooth scrolling navigation
- Mobile-friendly layout

## Structure

- `index.html` - Main website structure
- `styles.css` - All styling and responsive design
- `script.js` - Interactive functionality
- `README.md` - This file

## Deployment

### GitHub Pages

This site is ready to deploy on GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages
3. Select "main" branch as source
4. Click Save
5. Your site will be live at `https://dannykrikorian.github.io/lodestar-strategy/`

### Other Platforms

- **Netlify**: Drag and drop the repository
- **Vercel**: Connect GitHub repository
- **Traditional hosting**: Upload files via FTP

## Customization

### Update Stripe Integration

Replace the demo Stripe button in `script.js` with your real Stripe publishable key and payment link.

### Modify Content

Edit `index.html` to update:
- Products and pricing
- Contact information
- Company details

### Brand Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #ffffff;
    --accent-color: #c41e3a;
    --text-color: #333333;
    --border-color: #e0e0e0;
}
```
